GLPlus
======

Basics of a web-based app to allow a user to create, edit, delete items from a grocery list.

Initial design will be responsive so user can use on a mobile device as well.

v1.0
==============
setup basic architecture with jquery, require, backbone, underscore
<br>
Home Page with basic description and navigation
<br>
Implemented basic "todo" list in order to add/remove items to the list

Next Release
===============

<br>
